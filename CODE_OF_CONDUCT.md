# Code of Conduct

This project operates on respect, practical engineering, and clear communication.

## Expected Behavior

* Default to patience and assume good intent.
* Keep discussions anchored to code, documentation, and user outcomes.
* Deliver feedback constructively.
* Respect varying experience levels and communication styles.

## Unacceptable Behavior

We maintain a strict policy against:

* Harassment, threats, or personal attacks.
* Discriminatory language or conduct.
* Publishing private information without explicit consent.
* Persistent disruption after a maintainer requests it to stop.

## Enforcement

Maintainers enforce these standards. We will remove comments, close issues, or permanently ban contributors who violate them.

If you encounter an issue, contact a maintainer directly through the project repository.
